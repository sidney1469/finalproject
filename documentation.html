Project Metadata

Project Name: Tracking Aircraft for a Sustainable Future for Brisbane

Last Updated: 2026-05-11

Maintainers: Sidney Neil, Fiachra Richards, Xander Akison

Architecture / System Summary

The system aims to track Aircraft with the RTL2832U and record their location with great accuracy. This data will then be used to create a interface in which aims to help Brisbanites make more informed decisions on their home and living situation. To create a better Brisbane for a more Sustainable future.
